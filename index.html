Smart Air Quality Monitoring System

Real-time pollution tracking using IoT & automation

Introduction

This project uses IoT sensors and ESP32 to monitor air quality in real-time and control pollution automatically.

Problem

Traditional systems are expensive and lack real-time monitoring. Our system solves this with low-cost smart automation.

Solution

Sensors

Detect harmful gases like CO2 and smoke.

ESP32

Processes and sends data to cloud.

Automation

Controls fans/purifiers automatically.
